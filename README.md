# 💰 AI Finance Advisor

A full-stack personal finance management application with AI-powered insights, built with Java Spring Boot, React, and a Python FastAPI microservice integrating Groq's LLM API.

> A production-pattern finance app: track expenses and income, set budgets with real-time overspending alerts, work toward savings goals, and get AI-generated financial insights and a conversational AI assistant — all backed by a secure JWT-authenticated REST API.

---

## Features

### Core Finance Management
- Secure Authentication - JWT-based auth with BCrypt password hashing
- Expense Tracker - Full CRUD, categorized spending
- Income Tracker - Track multiple income sources
- Budget Planner - Set monthly category budgets with live overspending detection
- Savings Goals - Track progress toward financial goals with contribution history
- Dashboard - Live aggregated view of income, expenses, savings, and budget usage

### AI-Powered Features
- AI Insights - Structured, categorized financial insights (tips, warnings, praise, trends) generated by an LLM based on real spending data
- Spending Visualization - Interactive donut chart breaking down expenses by category
- AI Chat Assistant - Conversational assistant that answers questions like "Can I afford this?" using your real financial data

---

## Architecture

This project uses a three-tier, microservices-inspired architecture:
- Frontend communicates only with the Java backend
- Java backend owns all business logic, security, and data persistence, and delegates AI requests to the Python service
- Python AI service is a stateless microservice solely responsible for LLM prompt engineering and calling Groq

---

## Security Highlights

- Passwords hashed with BCrypt, never stored or returned in plaintext
- JWT-based stateless authentication with a custom filter chain
- IDOR protection - every resource lookup verifies ownership before returning data
- Global exception handling with consistent, safe error responses
- CORS explicitly configured, no wildcard origins in production config
- Environment variables for all secrets (API keys never hardcoded)

## Getting Started

### Prerequisites
- Java 17+, Maven
- Node.js 18+
- Python 3.10+
- PostgreSQL 14+
- A free Groq API key from https://console.groq.com

### 1. Database Setup
```sql
CREATE DATABASE finance_advisor_db;
```

### 2. Backend
```bash
cd backend
mvn spring-boot:run
```
Runs on http://localhost:8080

### 3. AI Service
```bash
cd ai-service
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
echo GROQ_API_KEY=your_key_here > .env
uvicorn main:app --reload --port 8000
```
Runs on http://localhost:8000

### 4. Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on http://localhost:5173
